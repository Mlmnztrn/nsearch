【実装機能】
・商品取り扱い店舗を都道府県単位で検索
・店舗の地図表示
・投稿（写真＋テキスト）機能


【制作理由】
世界的某店の某メニューが大好物です。
しかし、その商品の取扱店が多くはないことを
2020年2月に知りました。
（中略）
どの店舗で飲めるかを手軽に知りたい、
と思い制作しました。
